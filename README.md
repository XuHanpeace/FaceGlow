# 美颜换换（FaceGlow）

美颜换换是一款基于 AI 的人像创作应用，面向普通用户提供低门槛的“自拍 -> 风格生成 -> 保存/分享”体验。  
项目当前以 React Native 作为跨端基础，结合腾讯云开发与云函数能力，支撑多种图像/视频生成任务与会员、金币、任务激励等业务模块。

## 产品维度

### 产品定位

- 一键人像创作：用自拍快速生成不同风格内容。
- 模板化创作：以“专辑/模板”形式组织玩法，降低创作门槛。
- 连续使用激励：通过任务中心、签到与奖励机制提升活跃与留存。
- 商业化支撑：提供订阅与金币充值能力，覆盖不同用户付费路径。

### 核心能力

- AI 人脸融合：从模板市场进入创作页，选择自拍后触发融合生成。
- 多模型异步任务：支持图生图、图生视频、视频特效、人像风格重绘等任务链路。
- 作品管理：生成结果可预览、保存与分享，并支持历史作品回看。
- 用户体系：支持匿名使用、手机号验证码登录/注册、会话刷新与长期认证。
- 支付体系：集成订阅（RevenueCat）与应用内金币消费/充值流程。

### 用户下载与使用（非开发安装）

1. 在 iPhone 打开 App Store，搜索“美颜换换”。
2. 下载并打开应用后，可先以游客方式体验核心功能。
3. 进入首页选择喜欢的专辑模板，上传或拍摄自拍照。
4. 等待生成后，在结果页查看对比效果并保存/分享作品。
5. 需要更多权益时，可在个人中心或购买页开通订阅/充值。

> 说明：具体功能与活动内容会随版本更新而变化，请以线上版本为准。

## 技术架构维度

### 客户端架构（React Native）

- **入口层**：`App.tsx` 负责初始化导航、全局状态、监控、支付 SDK、更新能力与全局事件监听。
- **导航层**：`src/navigation/StackNavigator.tsx` 管理首页、创作流程、任务中心、个人中心、订阅与调试页面路由。
- **状态层**：`src/store` 使用 Redux Toolkit 管理认证、活动数据、异步任务、用户作品与任务进度。
- **页面层**：`src/screens` 按业务场景组织页面（首页、创作前页、结果页、任务中心、支付等）。
- **服务层**：`src/services` 抽象认证、数据库访问、云函数调用、支付、监控、存储缓存、分享等能力。

### 后端与云能力接入

- **腾讯云开发（CloudBase）**
  - 认证：手机号验证码、匿名登录、token 刷新。
  - 数据：通过数据库 API 管理用户、作品、活动等模型数据。
  - 云函数：用于人脸融合、异步任务发起与任务查询。
- **对象存储/媒体资源**
  - 模板图与结果图通过网络资源加载，配合缓存与预加载提升首屏与滚动体验。
- **异步生成任务链路**
  - 任务发起 -> 落库占位 -> 轮询/回查 -> 成功回填作品 -> 前端悬浮条/面板同步状态。

### 关键工程能力

- **网络与鉴权拦截器**：统一处理鉴权头注入、401 刷新、自动 UID 注入等逻辑。
- **可观测性**：接入 Aegis，覆盖页面访问、用户行为、接口错误与业务异常上报。
- **缓存与性能**：首页专辑与分类缓存、图片预加载、局部动效优化，减少等待与跳帧。
- **跨端原生扩展**：通过 TurboModule（`NativeFaceGlowTurbo`）预留高性能原生能力接入点。
- **热更新能力**：集成更新组件用于版本迭代发布策略（开发环境默认关闭更新策略）。

## 业务流程概览

1. 用户进入首页，加载专辑与分类配置（优先缓存，后拉新数据）。
2. 选择模板并上传自拍，进入创作页确认参数。
3. 发起云函数或异步任务，消费对应权益（免费/金币/订阅）。
4. 任务完成后回写作品数据并在结果页展示。
5. 用户可保存到相册、分享或在作品列表再次查看。
