隐私政策

最后更新日期：2025年10月14日

欢迎使用美颜换换（以下简称"本应用"或"我们"）。我们非常重视您的隐私保护和个人信息安全。本隐私政策将帮助您了解我们如何收集、使用、存储和保护您的个人信息，特别是关于面部数据的处理。

重要提示：在使用本应用之前，请您务必仔细阅读并充分理解本隐私政策的全部内容。如您不同意本政策的任何内容，请立即停止使用本应用。

1. 我们收集的信息

1.1 面部数据收集

本应用会收集您上传的包含面部信息的照片，用于AI换脸和美颜处理。具体包括：

  • 面部图像：您主动上传的自拍照片和人像照片
  • 面部特征数据：从照片中提取的面部关键点、轮廓等特征信息
  • 处理结果：经过AI处理后生成的美颜或换脸照片

1.2 其他信息收集

  • 账号信息：手机号码（用于注册和登录）、用户ID
  • 设备信息：设备型号、操作系统版本、设备标识符
  • 使用数据：应用使用时长、功能使用情况、操作日志
  • 订阅信息：订阅类型、购买记录、到期时间
  • 网络信息：IP地址、网络类型（WiFi/移动网络）

2. 面部数据的使用目的

2.1 核心功能服务

  • AI换脸处理：使用您上传的面部照片进行AI模型训练和换脸处理，生成艺术照片
  • 美颜效果：对照片进行美颜、美化处理，提升照片质量
  • 模板匹配：将您的面部特征与预设模板进行匹配，生成各种风格的照片
  • 效果预览：为您展示处理前后的对比效果

2.2 服务优化

  • 算法优化：使用脱敏的面部数据改进AI算法，提升处理效果和准确度
  • 质量提升：分析处理失败的案例，优化模型性能
  • 功能开发：研发新的AI功能和模板风格

2.3 用户体验

  • 历史记录：保存您的作品历史，方便您随时查看和下载
  • 个性化推荐：根据您的使用偏好推荐合适的模板
  • 技术支持：在您遇到问题时提供技术帮助

3. 面部数据的共享与第三方

3.1 第三方服务提供商

我们可能会与以下第三方服务提供商共享您的面部数据，但仅限于提供服务所必需的范围：

  • 腾讯云COS（对象存储）：
    • 用途：存储您上传的照片和生成的作品
    • 数据传输：采用HTTPS加密传输
    • 存储位置：中国大陆服务器
    • 安全措施：腾讯云提供的企业级安全保护
  • AI处理服务：
    • 用途：执行AI换脸和美颜算法
    • 处理方式：在安全的云端环境中处理
    • 数据使用：仅用于当次处理，不用于其他目的

3.2 我们不会做的事情

  • 不出售：我们绝不会向任何第三方出售您的面部数据
  • 不公开：未经您明确同意，不会公开展示您的照片
  • 不滥用：不会将您的面部数据用于广告或营销目的
  • 不转让：不会将面部数据转让给任何其他公司或组织

3.3 法律要求的披露

在以下情况下，我们可能需要披露您的信息：

  • 法律法规规定的情况
  • 司法机关或政府部门的要求
  • 保护我们或用户的合法权益所必需
  • 应对紧急情况（如人身安全威胁）

4. 面部数据的存储位置与安全

4.1 存储位置

  • 服务器位置：中国大陆（腾讯云服务器）
  • 本地存储：您下载的作品保存在您的设备本地相册
  • 临时存储：处理过程中的临时数据存储在云端

4.2 安全措施

我们采取以下安全措施保护您的面部数据：

  • 加密传输：使用HTTPS/TLS协议加密所有数据传输
  • 加密存储：对存储的照片和数据进行加密处理
  • 访问控制：严格限制员工访问权限，仅授权人员可访问
  • 安全审计：定期进行安全评估和漏洞扫描
  • 备份机制：定期备份数据，防止数据丢失
  • 防护措施：部署防火墙、入侵检测等安全防护系统

5. 面部数据的保留期限

5.1 保留时长

  • 上传的原始照片：
    • 自拍照片：在您主动删除前一直保留
    • 处理临时照片：完成处理后立即删除
  • 生成的作品：
    • 在您主动删除前一直保留
    • 账号注销后保留30天，之后永久删除
  • 面部特征数据：
    • 处理完成后24小时内删除
    • 仅用于当次处理，不长期保存
  • 用于算法优化的数据：
    • 经过完全脱敏处理后保留
    • 无法识别到个人身份
    • 保留期不超过2年

5.2 删除机制

您可以随时通过以下方式删除您的数据：

  • 在应用内删除单个作品或自拍照片
  • 在个人中心清空所有作品
  • 注销账号（将删除所有相关数据）
  • 联系客服请求删除（privacy@faceglow.app）

6. 您的权利

您对自己的面部数据和个人信息享有以下权利：

6.1 访问权

  • 查看我们收集和存储的您的所有数据
  • 了解数据的使用目的和方式

6.2 更正权

  • 更正不准确的个人信息
  • 更新您的账号信息

6.3 删除权

  • 删除您上传的照片和生成的作品
  • 请求删除所有个人数据
  • 注销账号并删除所有信息

6.4 撤回同意权

  • 随时撤回您对数据处理的同意
  • 停止使用特定功能

6.5 数据可携带权

  • 下载您的所有作品到本地设备
  • 导出您的个人数据副本

6.6 投诉权

  • 向我们投诉数据处理问题
  • 向监管机构投诉

7. 未成年人保护

我们非常重视未成年人的隐私保护：

  • 本应用面向18岁及以上的成年用户
  • 如果您是未成年人，请在监护人同意和指导下使用
  • 我们不会故意收集未成年人的面部数据
  • 如发现收集了未成年人数据，我们将立即删除
  • 监护人有权要求删除未成年人的所有数据

8. Cookie和类似技术

我们使用以下技术来改善用户体验：

  • 本地存储：保存您的偏好设置和登录状态
  • 分析工具：了解应用使用情况（匿名数据）
  • 崩溃报告：收集崩溃日志以改进应用稳定性

这些技术不会用于收集面部数据，仅用于应用功能和性能优化。

9. 隐私政策的更新

我们可能会不时更新本隐私政策。当我们对政策进行重大变更时，我们会：

  • 在应用内通过弹窗或通知提醒您
  • 在本页面顶部更新"最后更新日期"
  • 对于重大变更，征求您的明确同意

建议您定期查看本政策以了解最新信息。

10. 国际数据传输

  • 确保目标国家/地区有足够的数据保护水平

  • 遵守相关法律法规要求
